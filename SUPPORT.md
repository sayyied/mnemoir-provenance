# Support

Support covers documented local installation, released Python/CLI contracts, controlled adapters, and reproducible defects on supported Python versions. It does not include hosted operation, unmanaged access to private stores, custom framework adapters, legal clearance, data recovery beyond documented transaction scope, or production support commitments.

Before asking for help, capture the version, command, safe error code, operating system, Python version, and a synthetic reproduction. Never attach real memories, databases, credentials, absolute private paths, or transaction backups.
