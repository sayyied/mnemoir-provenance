# Public projection workflow

The repository is generated from a reviewed positive allowlist. Every output maps to source, template, or generator. The AST-based private exporter requires CPython 3.11; its runtime is bound into projection provenance and the source-candidate digest. Public contributions are reviewed normally; maintainers backport generated-core changes to canonical source/templates, regenerate twice under that runtime, compare, and preserve authorship.

The public tree has no access to private history. A generated diff never authorizes publication by itself.
